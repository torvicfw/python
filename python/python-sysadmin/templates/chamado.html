Criar Aplicação Intranet

Preciso que seja criado um repositório no Git chamado Intranet, e que os desenvolvedores Joaquim Lacerda e Rosemberg Farias sejam adicionados como desenvolvedores.

Ambos são desenvolvedores novos, então eles ainda não tem usuário criado. O email dos dois são joaquim.lacerda@dexter.com.br e rosemberg.farias@dexter.com.br

Preciso de 2 Containers em Docker para fazer o deploy da aplicação, um dos containers será utilizado para o deploy de produção e o outro para homologação.

A urls são, intranet.dexter.com.br e intranet-hom.dexter.com.br.

Preciso também que esse deploy seja feito de forma automatizada, quando for feito um PUSH na branch developer, o deploy deve ser realizado no servidor de homologação, e quando for realizado um PUSH na branch master o deploy deverá ser realizado no servidor de produção.
